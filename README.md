
Laser Ablation Propulsion for CubeSats Using Nanosecond Laser Pulses

Overview

This repository documents an independent experimental research project investigating nanosecond laser ablation propulsion for CubeSat applications using PTFE (polytetrafluoroethylene) as a solid propellant.

The objective is to determine whether nanosecond laser pulses with energies between 10 mJ and 100 mJ can generate measurable thrust under vacuum conditions and provide a simple, reliable, and low-cost propulsion concept for small spacecraft.

Research Objectives

- Measure thrust generated by nanosecond laser ablation of PTFE.
- Measure propellant mass consumption.
- Characterize plasma plume behavior.
- Estimate specific impulse (Isp).
- Compare experimental results with theoretical models.

Experimental Method

The experiments will be performed in a vacuum chamber using controlled nanosecond laser pulses directed onto PTFE targets. Diagnostic equipment will be used to measure thrust, plasma behavior, and propellant consumption.

Expected Impact

If successful, the project could demonstrate a practical laser-ablation propulsion system suitable for CubeSat attitude control and future small-spacecraft propulsion applications.

Project Status

Current stage: Experimental planning and repository preparation.
