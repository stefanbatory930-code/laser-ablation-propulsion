Project Specification

Project Title

Nanosecond Laser Ablation Propulsion for CubeSats Using PTFE

Purpose

The purpose of this project is to investigate the feasibility of using nanosecond laser ablation of PTFE as a simple, low-cost propulsion system for CubeSat spacecraft.

Propellant

Polytetrafluoroethylene (PTFE)

Laser

- Pulse duration: Nanoseconds
- Pulse energy: 10–100 mJ

Experimental Environment

- High-vacuum chamber
- Optical focusing system
- PTFE target holder

Measurements

- Generated thrust
- Impulse per pulse
- Specific impulse (Isp)
- Propellant mass loss
- Plasma plume characteristics

Expected Outcome

The project aims to determine whether nanosecond laser ablation can provide measurable thrust suitable for future CubeSat propulsion applications.

Current Status

Repository preparation and experimental planning.
